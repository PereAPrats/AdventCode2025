
# Advent of Code 2025

Este repositorio contiene mis soluciones a los retos de **Advent of Code 2025**.

## ¿Qué es Advent of Code?
[Advent of Code](https://adventofcode.com/2025/about) es un calendario de desafíos de programación que se publica cada diciembre. Cada día se desbloquea un puzzle con dos partes, diseñado para mejorar habilidades de programación, lógica y algoritmos.

## Estructura del repositorio
Cada reto tiene su carpeta **day<N>/** y dentro de cada carpeta hay dos archivos: **input.txt** con
los datos de entrada y **solution.py** con mi propuesta de solución.
